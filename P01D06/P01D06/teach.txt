Вот примерное деление:

Участник 1: Работа с игральным полем, ракетками и отрисовкой

Реализация игрового поля размером 80 на 25 символов.

Отрисовка ракеток размером по 3 символа и мяча символом.

Обновление графики после каждого хода.

Отвечает за вывод счета на экран.

Обеспечивает корректное отображение UI с учётом ограничений ASCII графики.
=====================================================================================

Участник 2: Логика управления и обработка ввода

Обработка клавиш управления ракетками: A/Z для первого игрока, K/M для второго.

Обработка пропуска хода по пробелу.

Обеспечение перехода программы в состояние ожидания ввода.

Валидация вводимых команд (только допустимые клавиши).

Интерфейс для переключения между ходами игроков.

=================================================================================

Участник 3: Игровая логика и правила

Реализация пошагового режима игры.

Управление позиционированием мяча (логика движения, отскоки от ракеток, границ).

Подсчёт очков и определение победителя при достижении 21 очка.

Отработка завершения игры с выводом поздравления.

Соблюдение структурного программирования по Дейкстре (разделение логики, отсутствие глобальных массивов и динамической памяти).
